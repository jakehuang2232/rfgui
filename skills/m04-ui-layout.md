# Layout System

## Principles

- no x/y positioning
- use container layout
- support flow + wrap

## Size

- width/height in style
- percent based on parent inner size

---

## Scroll Model

- ScrollDirection:
    - None / Vertical / Horizontal / Both

- handled via hit-test + bubble

## State

- scroll_offset
- content_size

## Scrollbar

- auto show/fade
- draggable thumb
- clickable track
